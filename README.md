## 실행 방법
python3.6(64bit) 버전에서 pip을 사용하여 flask 프로젝트를 수행할 가상환경을 세팅해놓아야 한다.
```python
mkdir flaskserver
cd flaskserver
conda create -n flaskserver python=3.6
```
가상환경 접속
```python
conda activate flaskserver
```
라이브러리 설치
```python
pip install -r requirements.txt
```
flask server실행
```python
python3 flaskserver.py
```
